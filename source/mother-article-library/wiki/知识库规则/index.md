# 知识库规则目录

最后更新：2026-05-05

本页是知识库规则分卷的导航页，方便人类阅读时点击跳转，也方便 Agent 按任务渐进读取。

## 分卷导航

- [[01-候选区总则]]：理念根基、知识库定位、核心原则、知识库分层、候选区、Schema 和操作流转。
- [[02-入库与来源管理]]：入库来源类型、来源置信度、账号评分、文章账本、账号与矩阵账本、OCR 与图片标准。
- [[03-产品与案例页面标准]]：产品母页、产品拆页、案例故事。
- [[04-创作框架与写作资产]]：叙事机制、读者转化路径、认知意外、写作调用流程、大师文章吸收、框架去重与组合。
- [[05-质量控制与风险治理]]：已明确标准、待明确标准、未来风险点和下一步建议。
- [[06-选题与搜索意图管理]]：SEO、微信搜一搜、豆包 / Kimi / DeepSeek / 元宝等 GEO 问题的选题来源、搜索意图和交易距离管理。

## 入口文件

- [[AGENTS]]：Agent 入口配置。任何 Agent 进入 `output_md/` 知识库母文件夹时，应先读该文件，再按任务读取对应分卷。
