# 新加坡宏利 Signature Legacy Harvest

## 1. 产品快照

| 项目 | 内容 |
|------|------|
| 保司 | 宏利新加坡（Manulife Singapore）——与香港宏利完全独立，不共享分红基金池、监管体系和产品设计 |
| 产品类型 | 美元计价分红型终身寿险（Participating Whole Life） |
| 计价货币 | 美元 |
| 缴费方式 | 趸交 / 3年缴 / 5年缴 |
| 最低保费 | 趸交10万美元；3年缴约3.33万/年；5年缴2万/年 |
| 投保规则 | 无需体检，流程简便 |
| 推出时间 | 2026年初 |
| 演示利率 | 5.25% / 7.5% 两档（注意：非实际IRR） |

## 2. 核心卖点

### 流动性：首日85%保证现金价值

趸交版本保单生效当日，保证退保价值即达已缴保费的 85%。传统终身寿险前 3-5 年现金价值普遍低于 30%，这个数字是行业领先。

### 回本速度：预期第5年回本

远快于传统产品 10-15 年的回本周期。

### 收益演示

- 7.5% 演示情境下，60年预期 IRR 约 6.45%
- 持有至100年，预期 IRR 接近 7%
- 重要：演示收益率为分红基金投资假设，需扣除佣金、管理等费用，实际 IRR 低于演示利率

### 传承功能矩阵

- 无限次更换受保人（生效满2年后）
- 第二被保人 + 后备持有人（双重保障传承连续性）
- 保单拆分（最多5份，适配多子女分配）
- 身故金分期给付（2-10年，防挥霍/婚变/债务）

### 健康关怀

- 重症关怀：确诊指定重症可提前提取100%终期红利
- 末期疾病提前赔付：生存期不足12个月可预支身故保险金

## 3. 资产配置结构

- 固定收益：25%-55%（底仓）
- 公共股权 + 私募股权：45%-75%（增长引擎）
- 房地产等另类资产（抗通胀）
- 96%债券为投资级，70%达A级及以上

## 4. 核心缺陷与风险

### 非保证收益波动
年度红利、终期红利均非保证，极端市场环境下可能低于演示预期。无历史分红实现率数据可参考（新产品）。

### 早期退保仍有损失
虽首日85%保底，但短期退保仍亏本金，建议至少持有至第5年。

### 演示利率口径差异
5.25%/7.5% 两档演示，与香港市场通用的单档演示（如6.5%）口径不同，直接对比数字会产生误导。

## 6. 写作抓手

### 最该先抓住什么误会
"7.5%演示利率 = 实际能拿到7.5%收益"——这是最大的认知陷阱。演示利率是投资假设，扣完费用后实际IRR远低于此。

### 最容易写偏到哪里
- 写偏方向1：把"首日85%"写成"随时可退保不亏钱"
- 写偏方向2：把新加坡 vs 香港的数字直接对比，忽略演示口径差异
- 写偏方向3：把"无限次换受保人"写成独家功能（香港多款产品也有）

### 适合调用的表达动作
- 把"首日85%"翻译成"结构性流动性保障"而非"退保承诺"
- 把两档演示翻译成"乐观/中性两种假设下的能力边界"
- 把新加坡定位翻译成"香港之外的第二张牌"
