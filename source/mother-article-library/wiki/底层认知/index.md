# 底层认知

最后更新：2026-05-13

本目录统一承接原先容易重叠的“领域总纲 / 主题与风险 / 产品形态”三类内容。

它不是产品母页、保司母页、案例故事或创作框架的替代品。这里存放的是写香港储蓄险文章时反复调用的底层判断、产品形态谱系、收益结构、配置逻辑、宏观风险和高价值漏洞。

## 入库边界

- 跨产品、跨保司、跨场景复用的底层判断，进本目录。
- IUL、保费融资、万用寿险、年金转换、养老社区权益等形态级概念，先进 `[[产品形态谱系]]`。
- CRS、离婚、税务、美元配置、监管变化等宏观或高客风险，如果能反复服务创作，也进本目录。
- 单一产品事实回 `wiki/产品母页/`；单一保司事实回 `wiki/保司母页/`；单一故事回 `wiki/案例故事/`；文章叙事套路回 `wiki/创作框架/` 或 `wiki/候选区/`。
- 底层认知页必须优先成为“写作调用单元”。短于 2000 字的主题默认并入已有聚合页，不直接占用正式页面。
- 只有当某个底层认知页长到单页难维护，且子主题能独立服务多类写作时，才从该页拆出同目录子页；禁止提前建立空壳分类。

## 当前页面

- [[港险基础认知与计划书阅读]]：港险总纲、计划书阅读、保证 / 非保证收益、币种与汇率决策的合并调用页。
- [[跨境投保与保单架构]]：赴港投保流程、保单角色结构、资产隔离的合并调用页。
- [[CRS与香港保险的税务边界]]：CRS申报机制、税务居民身份、保单分红与理赔的税务处理、早期零现金价值的规划意义、CRS 2.0变化。
- [[产品形态谱系]]：产品形态、机制类型与具体产品 / 具体成交方案之间的路由判断。
- [[搜索意图与高价值词根映射模型]]：SEO、微信搜一搜、GEO、成交味排序、搜索型内容打法和选题承接模型。
- [[分红实现率判断模型]]：分红实现率、总现金价值比率、保证与非保证之间的判断顺序。
- [[高价值缺陷与漏洞]]：跨产品、跨场景复用的高价值反向判断、行业漏洞和销售话术漏洞。
- [[港险产品-用户需求反查表]]：扁平数据库表，按产品名列出适用人群和希希评论，供 Agent 在文章结尾调用做破窗引导。
