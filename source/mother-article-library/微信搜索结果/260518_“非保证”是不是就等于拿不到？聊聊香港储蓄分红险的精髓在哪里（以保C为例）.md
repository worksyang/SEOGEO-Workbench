# “非保证”是不是就等于拿不到？聊聊香港储蓄分红险的精髓在哪里（以保C为例）

丽娟在香港_2026年5月16日 15:08__中国香港_

做港险这么多年，有两个问题，我几乎每次都要被客户问一遍：

非保证部分就是拿不到吗？你说的复利5%-6%，真的能实现吗？

我非常理解这种顾虑，毕竟习惯了看内地的保证1.75%-2%，甚至之前有保证3.5%。“非保证“三个字，脑子里已经自动翻译成了：“那就是你说了算，想给就给，想不给就不给”。说实话，这是最大的误区。

我先说结论：

非保证≠ 拿不到，只是不把数字写死。

你要担心的不是“有/没有”收益，而是未来收益“多/少”。

下面我从投资逻辑、历史兑现、监管披露三个角度做个详细拆解。

# 01 香港储蓄险的精髓，就在于非保证分红

香港储蓄险收益通常分为两部分：

🔹 保证现金价值

保证回本、本金不受损失，负责给你一个安全底线。长期收益也就1%左右，单纯靠保证部分，连通胀都跑不赢，意义不大。

🔹 非保证收益

收益的大头，整份保单能不能复利增值、能不能拉开差距，全靠这一块。它的本质是分红靠投资赚钱，市场有涨有跌，资产会波动 → 回报不可能年年一个数 → 就不能写成保证（刚性兑付）

🔹 内地VS香港，投资逻辑核心差距

保险公司投资类别跟内地差不多，债券、固收、股票等多元资产配置，同时搭配私募股权、基建等另类投资。

最大区别在于投资范围：境内资管只能投境内（政策限制），香港的资管可以全世界范围内投资，哪儿有机会投向哪里。

保险公司拿着客户的钱，大多数也是交给了全世界最牛逼的资产公司帮忙管理，如保C长期合作的贝莱德、黑石、KKR等，加上保C资金池庞大，超过了1500亿美金，也能拿到低费率，和散户投不到的很多优质项目。


![图片](https://mmbiz.qpic.cn/sz_mmbiz_jpg/cXNib8XM7AeRNbdMK2KVVtIL4Tpaf2ENtXzPRhxQEUcq4PMz1MnaIlvOC9ga49UPmCGtH8gzhibAmztgBwiaYcEgtS1Hwg0gRjXcyd2IhJeCIg/640?wx_fmt=jpeg&from=appmsg#imgIndex=1)


短期市场肯定有波动，但拉长时间看，数据不会骗人：

10年期的美国国债过去30年年化收益约4%

标普500指数，过去40年翻了10倍，平均年回报率11%

这样的股债平衡基金，搭配专属“缓和调整机制”，降低极端情况下收益大幅波动风险，长期持有，给客户5%-6%的复利收益，完全在合理区间。

所以，“低保证+高非保证”反而是更聪明的设计。保证部分用来兜底，非保证部分用来赚钱。如果只盯着那1%的保证，等于主动放弃了资产上涨的天花板。

# 02 不讲空话：历史真实分红数据

用过去看未来。正如我们家长给孩子选学校，升学率长期稳定的老牌名校，我们才更有信心，相信他未来可以做得到。

保C是全港唯一一家，主动公开了20年以上老保单的完整真实收益；以一张1996年的实际保单为例，过往30年平均年度IRR6%，总回报512%。


![图片](https://mmbiz.qpic.cn/mmbiz_jpg/cXNib8XM7AeSxtpZClniamSGqp5rbXKEdhMib7faLH8HJediamTcFIP5GFyiamkyYRw9jn54icWjyVFEjg3YELmCGyFRzMnXc560k6rvicVhwtuBe8/640?wx_fmt=jpeg#imgIndex=2)

说到底，买香港分红险不用纠结某一年的收益高低，真正值钱的，是几十年如一日的兑现能力。


# 03 监管披露：看懂分红实现率

香港保监局严格监管，在GN16规则下，所有保险公司都必须披露你过往产品的分红实现率，数据官网可查，无法造假。

实现率=实际到手总现金价值÷当初计划书演示的总现金价值。

如果实现率是90%，那么当时合同数据写100万，你到手就是90万。

注意，这里要看的是总现金价值实现率，而不是某一两个小指标的实现率，因为这才是你真实到手的总数。

以前被很多人跟风黑的隽S，哪怕市场最差的时候，实现率也是在80%。所以我经常跟我客户说：最差打个8折，如果你能接受这个波动，那就可以安心持有了。

近几年推出的隽F，信S明天等储蓄分红产品，实现率都在100%及以上，收益足额兑现。

04 写给所有纠结的人


非保证≠拿不到，只是不把收益锁死，保留上涨空间；

港险真正的价值，就在非保证分红；死守保证，等于放弃复利；

判断一款港险靠不靠谱，别纠结一句“保证/非保证”，重点看三件事：


1、投资逻辑是否讲得通（股债平衡、全球分散）；

2、监管披露（实现率）能不能验证兑现水平；

3、长期历史表现是否稳定（看长期，不看一年）。
