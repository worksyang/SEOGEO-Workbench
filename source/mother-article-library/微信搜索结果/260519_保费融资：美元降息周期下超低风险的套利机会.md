# 定义我们的不是说了什么而是做了什么

# 文：William

最近美联储连续降息，从9月至今已三次下调利率，累计达75个基点。这一轮宽松周期，让香港市场一个原本只在高净值圈层流传的玩法——**保费融资**，重新活跃起来。

所谓保费融资，逻辑其实和早年“用银行的钱炒房”很像：你只需拿出一小部分自有资金作为首付，其余大部分保费由银行提供贷款，买入一份大额保单。如果保单本身稳健增值，再叠加杠杆效应，整体回报就会被显著放大。简单说，就是“用便宜钱，撬动高价值资产”。

当前最热门的组合是**中银人寿保单 + 集友银行融资**。这套方案之所以抢手，是因为它同时满足两个核心条件：一是保单本身够硬，二是融资结构足够高效。

先看保单端。以一份150万美元的保单为例，目前正值保险公司“开门红”促销期，可享10%折扣，实际只需135万美元即可上车。更关键的是，这款产品的预期表现亮眼：预计3年回本，5年总回报率达5.04%，在同类储蓄型保险中属于第一梯队。也就是说，即便不融资，单靠保单自身收益也相当可观。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/AGaQibvIHQLtxyzEXh8bic83ibTicQIn0FSDE8RzQR878FGDKmoiaOo0mUh21sdWj5JXBLnWrcJmzhfGjQ15H1XJbLg/640?wx_fmt=png&from=appmsg&tp=wxpic&wxfrom=5&wx_lazy=1&watermark=1#imgIndex=3)

再看融资端。集友银行提供的贷款比例高达保单首日现金价值的95%，这意味着杠杆可以拉到**7.55倍**。换算下来，你只需自掏不到20万美元的首付，再支付约1.15万美元的手续费，就能拿下这份135万美元的大额保单。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/AGaQibvIHQLtxyzEXh8bic83ibTicQIn0FSD7znOm7icEtrTafOxdYv9C03fkOuFpIofq1aMDcUnUiaohjphwGLLemicA/640?wx_fmt=png&from=appmsg&tp=wxpic&wxfrom=5&wx_lazy=1&watermark=1#imgIndex=4)

而贷款利率采用香港P（Prime Rate）减1.9%，目前约为3.35%。在这样的成本下，整套方案测算出的**5年退出年化收益率最高可达13.63%**，远超市面上绝大多数稳健型金融产品。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/AGaQibvIHQLtxyzEXh8bic83ibTicQIn0FSDwzuV7lDeRmzE531KzTpCr9LFgoxznH4oJicuQ5dZGe35HThSSvxibjyQ/640?wx_fmt=png&from=appmsg&tp=wxpic&wxfrom=5&wx_lazy=1&watermark=1#imgIndex=5)

为了验证其抗风险能力，可以做两组压力测试。
第一种情况：假设未来利率上升至4.5%（高于当前水平），且保单分红100%实现——这已是相对不利的环境，但5年年化收益仍能维持在**7%左右**，在当下低利率时代依然极具竞争力。


![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/AGaQibvIHQLtxyzEXh8bic83ibTicQIn0FSDN0GeenAN74h2n1msNGgyTv7VSU5NvI6EP7Nxp5eHuYtyDAVsGSibFmA/640?wx_fmt=png&from=appmsg&tp=wxpic&wxfrom=5&wx_lazy=1&watermark=1#imgIndex=6)

第二种情况：假设分红实现率降至70%（极端悲观情景），但利率保持3.35%不变，整体收益仍然为正。考虑到中银人寿过往6–10年期保单的平均分红实现率稳定在95%上下，这种极端情形发生的概率极低，足见该方案的容错空间相当宽裕。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/AGaQibvIHQLtxyzEXh8bic83ibTicQIn0FSD23ezWkmcYTVVDcCC62ZjGpL8QerJofliaYXYicFq3pqXy7sHVIY0HJyw/640?wx_fmt=png&from=appmsg&tp=wxpic&wxfrom=5&wx_lazy=1&watermark=1#imgIndex=7)

当然，参与保费融资并非零门槛。首先，银行要求最低贷款额为100万港币，折合下来首付大约3万美元起；其次，需提供与总保费等额的流动资产证明——比如存款、股票、基金等，房产、私募或信托不算。好消息是，集友银行接受内地出具的资产证明，对跨境客户较为友好。

而保费融资最大的风险还是在于分红实现率是否能达到预期，根据过往中银的分红水平，第三方的数据统计显示：中银生效 6–10 年以上的中银保单，平均分红实现率95%。在香港分红险里，算是非常稳的第一梯队（这里也特别补充一句，做保费融资千万要避开分红实现率表现不稳的公司，否则对于收益还是现金流都将是一个极大的考验，香港保监局为此进行过多次风险提示）。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/AGaQibvIHQLtxyzEXh8bic83ibTicQIn0FSDJibv1HnyaTZ5eRXtBQEShdvsMptRqGoCy228cAh8Zz4AZKLMbPMNBxQ/640?wx_fmt=png&from=appmsg&tp=wxpic&wxfrom=5&wx_lazy=1&watermark=1#imgIndex=8)

需要提醒的是，这类高杠杆、高效率的融资额度并非无限供应。眼下正值美联储降息周期、香港P利率处于历史低位、保险公司又有开门红优惠，三重利好叠加，导致预约已排到十天之后。对于有充足流动资产、具备一定金融认知的非小白投资者来说，这确实是一个值得关注的时间窗口。


![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/AGaQibvIHQLtxyzEXh8bic83ibTicQIn0FSDAAkInpy3O5t0ALIOIjYejOyy1GKa6X4gQ0y6sNVvkaB77Y3hBkSNUw/640?wx_fmt=png&from=appmsg&watermark=1#imgIndex=6)


本质上，保费融资不是投机，而是在特定宏观环境下，利用制度性利差和优质底层资产，构建的一种**安全边际较高的杠杆配置策略**。它适合那些理解风险、重视确定性、并愿意用时间换复利的人。当“便宜钱”再次回归，聪明钱早已开始行动。
