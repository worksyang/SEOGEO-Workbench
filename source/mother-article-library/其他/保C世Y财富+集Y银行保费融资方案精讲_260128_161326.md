# 什么是保费融资？

![](images/d6e90ac0a8a73ddf4600fe5a7b890a9daf6d74f3e48c66d6fdf1fa27032ac4dd.jpg)


# 定义

保费融资是一种利用银行贷款来支付保险保费的金融方案。 

![](images/c6d83a266d035714a25126726141043385826b2153f2301c40c6d111287b0358.jpg)


# 核心逻辑

客户支付部分首期保费，剩余由银行贷款支付，保单现金价值作为抵押。 

![](images/49e41724e157a7796b6795f5f06b79a1257084d9ee299b9ea01a945af4968ea3.jpg)


# 核心优势

杠杆效应：以小博大，放大投资回报。 

√ 资金灵活：释放现金流，用于其他投资或消费。 

√ 财富传承：大额寿险融资，优化资产配置，实现有效传承。 

![](images/44a5010006f19c3f4465c15993ae46d7c71f2b3a25b0158d589d614ace45c710.jpg)

# 常见问题答疑 (Q&A)

Q1：客户担心10年后底层保单增值达不到67%怎么办？ 

A1: 67%是基于100%分红实现率的测算，保单设有保证现金价值作为核心安全垫，如果按分红实现率为85%进行测算，持有满10年退出的IRR从8.39%降为6.71%，按首付金额计算的单利为从20.83%降为15.34%。 

Q2：集Y银行的贷款可以提前还款吗？ 

A2：可以。银行不设罚息期，但需每年重检。年检要求：提交余额不少于未来一年利息的常用银行账户月结单作为供款证明。 

Q3：保单除了美元，支持其他币种吗？ 

A3：世Y财富保单目前仅支持美元，贷款货币建议选择港币 

Q4：资产证明的要求是什么 

A4: 按折扣后实缴保费计 1:1 的流动资产证明（签单前1个月内，包括现金、定存、开放式基金、股票及债券等；不计算房产、信托产品及封闭式基金；资产计算以审批为准）注：不接受时点证明，若提供汇总式资产证明，需提供资产明细及近1个月交易记录。 

# 增长潜力：港元最优贷款率与美联储利率的利差空间

![](images/c5d9c601a19ba2dc696c2f3ef9a89e51c07c176e57a9fc9e537875f315b689e7.jpg)


从历史数据看，美联储在2004-2006年进入加息周期时，由于香港银行结余较少，仅约40-50亿港币，香港最优贷款利率需要紧跟美联储利率的变动。 

但2008年金融危机后，美国量化宽松救市，大量资金流入市场，所以在2016-2019年及2022-2024年的加息周期中，香港最优贷款利率的调整往往更为滞后且加息幅度更小，这种时差与幅度差带来的利差，让保费融资在特定周期下具备了显著的收益空间，也为高净值客户提供了运用杠杆提升保单整体回报的机会。 

# 增长潜力：高银行总结余，跨境资金持续流入香港

![](images/df43ddb34ebc37009845372262da01e4903ae0e6f1f875799c6cd168dc18dad7.jpg)


香港金管局的「银行总结余」，是香港银行体系存放于金管局的结算户口总余额，本质是银行体系的「基础流动性池」，直接决定市场短期流动性供给。 

总结余越高，银行体系「闲置资金」越多，同业借贷成本（如HIBOR）越容易下行； 

总结余越低，银行间资金紧张，
借贷成本趋升，甚至可能引发流动性收紧预期。 

银行总结余与港元同业拆息率（HIBOR）呈「反向关系」，是预判融资成本的关键指标。 

# 增长潜力：港元融资利率区间稳定，融资利差持续

![](images/331ec501508fa8856d9eeded32f92bcaa9fcf5dc7d0b0cfe8e2e3c82e4c87e47.jpg)


在联系汇率制度框架下，港元融资利率维持 3%-4% 区间，与美元形成稳定的息差。 

这一优势利好美元计价保单的港币贷款保费融资业务：港币贷款不仅可以降低客户融资成本，更与美元保单的中長期收益形成正向利差空間，提升業務吸引力。 

稳定的港元利率环境也让保费融资规划更可控，减少融资成本的波动风险，有效带动保费融资业务规模增长與稳健運行。
