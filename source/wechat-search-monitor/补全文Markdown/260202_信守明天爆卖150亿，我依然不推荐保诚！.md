# 信守明天爆卖150亿，我依然不推荐保诚！

链接：https://mp.weixin.qq.com/s/jyKSlDAJ-ibKILD4PkI78A


最近保诚公布了一个数字。

去年2月才上市的信守明天，距今不到一年时间，单款产品狂揽150亿港元总保费。

![图片](https://mmbiz.qpic.cn/mmbiz_png/42Raen3ejpNWZgmgEkaGXEDfX7zX3Ic1L3tHNTvrzyniaiau9oxdRYowNbfTa6dNJltU3ouKHruULyD3VZDcNyXA/640?wx_fmt=png&from=appmsg)

这在内卷到极致的香港保险市场，可以说表现非常亮眼了。

并且最新的2025Q3季度的数据显示，保诚的标准保费收入依旧市场排名前列。

![图片](https://mmbiz.qpic.cn/mmbiz_png/42Raen3ejpNWZgmgEkaGXEDfX7zX3Ic1fqvk0HCDhR4sIniceIh6Cs2cWaxBZmBKzibynCEd9Ayv9liaZzRELd0fg/640?wx_fmt=png&from=appmsg)

看起来形势一片大好，完全没有受到这两年分红实现率波动大、红利回撤等负面新闻影响。

但是就我个人来说，我依然不会推荐保诚这家公司。

![图片](https://mmbiz.qpic.cn/mmbiz_gif/42Raen3ejpMc7NXLEAeku4bib0mhVQovVBk0eLBMNcp8rObmA2XVica8WpxgOxkavyI5dI155PIONrDfBJFZdq7w/640?wx_fmt=gif)

**首先，咱们先聊一下，信守明天这款产品，到底有什么优势？**

客观来说，产品确实不错。

**一方面，收益确实能打。**

5年缴费，最快28年，复利能达到6.5%；

![图片](https://mmbiz.qpic.cn/mmbiz_jpg/42Raen3ejpNWZgmgEkaGXEDfX7zX3Ic1nYK5iakAReTepokBJe6AHDHiahuW9o6Q714siadkOQsECxibvKaXjfg3IQ/640?wx_fmt=jpeg&from=appmsg)

这个到达封顶收益的速度，数一数二。

友邦，安盛的王牌产品都要30年。

**另一方面，功能非常有诚意，而且有很多创新。**

![图片](https://mmbiz.qpic.cn/mmbiz_png/42Raen3ejpNWZgmgEkaGXEDfX7zX3Ic1Mpia4fo4N6AlrSOVbNVHcviaRIiaWsPtLyjr6LeOQMBia5Sl9eXbynriaLg/640?wx_fmt=png&from=appmsg)

**第一个是，真多种货币自由切换**

很多保险公司的货币转换更像是给你换一张新保单，预期收益分红结构可能都会改。

但是信守明天可以说是目前市场上唯二真货币转换。

它支持美元港元人民币等6种货币自由切换，转换后还是原来那张保单，功能、特点、结构不会改变。

**第二个，是自主入息＋自主传承。**

自主入息的意思是，要求保司定时定额，从保单里给指定账户打钱，

不需要经过自己账户，不留痕，方便保护隐私；

自主传承可以在生前安排好身故赔偿的方式。

![图片](https://mmbiz.qpic.cn/mmbiz_png/42Raen3ejpNWZgmgEkaGXEDfX7zX3Ic1NwtkN7oaH6r1Yugx3Kibf48alMQammrSkXO3zicRmkKr2wesLibhUic6MQ/640?wx_fmt=png&from=appmsg)

比如，可以设定在孩子结婚时给一笔钱，买房时给一笔钱，控制力很强。

这两个功能现在香港遍地都是，但保诚属于首创，很有创造力。

![图片](https://mmbiz.qpic.cn/mmbiz_gif/42Raen3ejpMc7NXLEAeku4bib0mhVQovVBk0eLBMNcp8rObmA2XVica8WpxgOxkavyI5dI155PIONrDfBJFZdq7w/640?wx_fmt=gif)

那这款产品值得买吗？

说句大实话，产品是有优势，但是它的优势不足以弥补缺点。

所以，我认为大部分人都不适合！

为什么？

保诚有两个比较致命的问题，已经是众所周知了。

第一，**分红实现率波动大到离谱****，根据保诚自己公布的数据。**

2024年终期红利最高1044%，最低只有3%；

复归红利最高158%，最低仅13%；

![图片](https://mmbiz.qpic.cn/mmbiz_jpg/42Raen3ejpO8Zn0v7eMvmnxuahjIaUhLfibFwejaP6Oh6LG9u9kicCqNicTkLszbjkQeEibc63qPfF8Slk1T8c8wSQ/640?wx_fmt=jpeg&from=appmsg)

而且这可不是个例，他们24年公布了44款产品的276个分红实现率数据。

其中有将近一半的数据分红实现率低于80%。

**第二，出现了红利回撤，到手的钱比起前一年不多反而少了。**

出现这样的问题，底层原因有两个：

第一个是投资组合过于激进，信守明天这款产品的股权投资占比高达70%，并且是固定的，不能根据市场做动态调整。

![图片](https://mmbiz.qpic.cn/mmbiz_jpg/42Raen3ejpNWZgmgEkaGXEDfX7zX3Ic1K0277pIrL073h6RvCAIlZiatfg8pBOKl10icw61T8HaVicibGuLicpdibK1w/640?wx_fmt=jpeg)

对比一下其他家，比如友邦、国寿，股债配置通常是浮动的。

市场差就减仓，市场好就加仓，更灵活。

保诚这个股债占比，相当于把客户的储蓄险当股票基金在运作。

第二个是分红平滑机制做得不好。

当**然也有可能是分红到了平滑机制都很难挽救的地步了。**

看我账号的朋友，甚至去买港险的内地朋友，其实很大一部分都是港险小白。

大家来买保险，图是的什么？

是对冲风险、做长期稳健的资产配置、是给孩子做教育规划、给自己做养老规划，等等。

要的是安全、现金流、和稳定。

你体验的玩股票的刺激，结果挣了个保险的收益，图啥呢？

明明市场还有很多更好选择，有分红更好、收益同样高、公司背景也很强的，

为什么要去赌一个高波动、高不确定性的产品呢？

**保险保险、是转移风险、不是制造风险！**

![图片](https://mmbiz.qpic.cn/mmbiz_gif/42Raen3ejpMc7NXLEAeku4bib0mhVQovVBk0eLBMNcp8rObmA2XVica8WpxgOxkavyI5dI155PIONrDfBJFZdq7w/640?wx_fmt=gif)

当然，公正地说，也有少部分人可以去考虑这款产品。

第一，**你是一个真正长期主义的富豪，本身手握大额资产。**

香港保险只是做全球资产配置的一小部分。

不靠这部分来短期收益变现，更多的是做风险对冲和税务不确定性。

对这群人来说，短期分红波动，不过就是账户里跳动的一个数字，不影响整体财富结构。

保诚177年历史，以及香港保监局认证的大而不能倒的监管背书，确实值得你为它投票。

![图片](https://mmbiz.qpic.cn/mmbiz_png/42Raen3ejpNWZgmgEkaGXEDfX7zX3Ic1UNpMjNGvDW8c5lmicfU6pr5YGH00TtPkWmnHwKJYugERGPotz0LianGQ/640?wx_fmt=png&from=appmsg)

第二类是，高风险偏好玩家，想要追求超额收益。

这款产品在经济环境很好的时候，是有可能会带来远超预期的收益。

因为它的股权类资产足够多。

向上的弹性会比较大。

比如2024年，保诚就有产品分红超1000%的达成。

**对于大部分人来说，香港保险的演示收益都是没有意义的，宣传声量也并不代表什么。**

**关键是搞懂一家公司、一款产品的底层逻辑，根据自己的需求，合理选择。**
