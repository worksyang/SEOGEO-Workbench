# TSO 详情页字段表

> 基准入口：`/TsoKeyWordDetail/detail?keyword=`

## 1. 二级页总览

| 二级页 | URL模式 | 已确认字段 |
|---|---|---|
| 详情 | `/TsoKeyWordDetail/detail?keyword=` | `搜索指数 / 下拉词数量 / 下拉词月搜索 / 字数 / 国家 / 下拉词` |
| 相关词列表 | `/TsoKeyWordDetail/competitive?keyword=` | `关键词 / 搜索指数 / 字数 / 操作` |

## 2. 详情页

| 区块 | 字段 |
|---|---|
| 顶部卡片 | `搜索指数 / 下拉词数量 / 下拉词月搜索 / 字数` |
| 国家维度 | `国家` |
| 下拉词表 | `关键词 / 搜索指数` |

## 3. 相关词列表

| 字段 | 说明 |
|---|---|
| 关键词 | 相关词文本 |
| 搜索指数 | 相关词热度 |
| 字数 | 词长 |
| 操作 | 详情入口 |

## 4. 结构结论

TSO 详情页当前是全站最轻量的详情模型之一：

1. 没有 DSO 那种作品、电商、监测扩展层。
2. 重点差异不在二级页数量，而在 **国家维度**。
