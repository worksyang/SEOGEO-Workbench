## 第一部分：方法论与 7 日精确数据采集链路

### 1.1 真实 7 日数据的获取路径（✓ 数据支撑）

V1 报告只能看到 AIDSO 列表里的"月覆盖人次 / 平均:XXX"。V2.1 发现 AIDSO 的"7 日搜索人次"列是迷你 ECharts 折线图，**用真实鼠标 hover 在 7 个数据点 marker 上可以拿到每天的真实数值**。

具体采集步骤：
1. `navigate https://dso.aidso.com/KeywordDouyin/searchWord?keyword=XXX`，等待 5-8 秒
2. `evaluate` 给所有 marker path 加 className `c{chartIdx}-d{dayIdx}`
3. `browser_hover` 依次 hover `.c0-d0` 到 `.c0-d6`
4. 每次 hover 后 `evaluate` 读取 `div[style*="z-index: 9999999"]` 的 textContent

完整方法论已写入 010 文档第 24 章。

### 1.3 三大 V2 反常识（基于真实 7 日数据，✓ 数据支撑）

**反常识 A：AIDSO 显示的"平均"对万级以上高量级词做了向上加权，对低量级词是真实算术平均。** 试管婴儿主词显示 6880 实际 6635（差 3.6%）。其他 5 个样本都吻合。机制推测：AIDSO 用某种加权算法让大词看起来热度更高，吸引用户投流。**对实际业务影响：监控系统的"平均"指标不能直接用 AIDSO 显示值，需要自算。**

**反常识 B：月覆盖大 ≠ 当前活跃度高。** 全红婵戴劳力士月覆盖 174.61w，但 7 日搜索只有 0-3 次。这是一个已经过气的新闻事件词——新闻已经发生很久，过去 30 天累积搜索巨大，但当前活跃度归零。**对业务影响：仅看 AIDSO 月覆盖做内容选题会被严重误导，必须用 7 日数据过滤"还活着的词"。**

**反常识 C：7 日曲线出现 1 天尖峰 = 当天有视频爆了。** 试管婴儿"长大后问题"在 06-05 出现 13,205 的尖峰（平时 2000-3000）。这是"看后搜"杂音的可量化信号——尖峰当天一定有一条相关视频爆了，触发了大量搜索。**对业务影响：7 日数据可用于反向追踪爆款视频（找尖峰那天查热门内容）；种草型内容投放的最佳时机是"尖峰刚结束、流量还在高位"的 1-3 天。**

⚠ **章节末尾待验证**：尖峰是否总能对应一条爆款视频？需要选 5-10 个真实尖峰案例反查当日热门视频做交叉验证。

### 2.3 五种土壤类型的详细判断（✓ 数据支撑 + 部分 ⚠ 待验证）

**类型 1：真高成交意图（私人银行是唯一）**
- 私人银行是 20 个领域里**唯一**商机词占比 ≥ 15% 的（67%）。
- 客单价 600w+，抖音上没有任何其他品类能达到这个成交意图密度。
- ⚠ **待验证**：这个 67% 是 AIDSO 列表的前 20 词，扩展到前 100 词是否还能保持高比例？

**类型 2：种草大品类（试管婴儿 / 三代试管 / 国际学校 / ICL）**
- 月覆盖 ≥ 10w 但商机词占比 < 5%。流量大但成交意图弱。
- 适合做长期内容种草，导流到私域成交。

**类型 3：事件泡沫型（劳力士 / 长江商学院 / 南极旅游 / 私人飞机 / 家族信托）**
- 月覆盖被单一事件大幅推高（如劳力士 174w 是全红婵戴表新闻）。
- 7 日数据是过滤"还活着的词"的唯一可靠方法。

**类型 4：地域/IP 矩阵型（迪拜房产 / 高端婚礼 / 香港保险）**
- 没有强势主词，但个人 IP（玛丽/湃队/姚凯飞）矩阵化。
- 新进入者很难分流量，但 IP 本身有私域价值。

**类型 5：入口词错位（早癌筛查 / 高端体检）**
- AIDSO 上显示小品类，但换口语化种子词后词库暴增。
- **早癌筛查 → 胃肠镜**：60.20w 月覆盖、2180 个相关词、1824 个蓝海词、商机词密度 5%（胃肠镜多少钱 3.18w、胶囊胃肠镜价格 1.99w）。
- **高端体检 → 全身体检**：33.12w 月覆盖、981 个相关词、商机词密度 25%（5/20 是商机词）。
- 这两个品类应该从"荒漠"重新归类为"种草大品类"。

**类型 6：真荒漠（海外辅助生殖 / 干细胞回输）**
- 海外辅助生殖 0 词，美国试管 23 词，泰国试管婴儿 0 词，格鲁吉亚/俄罗斯相关未测试。
- 干细胞回输 27 词，9704 月覆盖，虽然比"免疫细胞存储"16 词 2027 大，但仍算小品类。
- ⚠ **待验证**：海外辅助生殖的格鲁吉亚/俄罗斯/哈萨克斯坦等具体国家词是否仍是荒漠？

⚠ **章节末尾待验证**：本评分矩阵的阈值（5%/15%/30%）是经验值，需要至少 30 个真实成交案例验证。

## 第五部分：本报告的局限性 + 未验证假设清单（前置）

按读者反馈把这一节提前。读者在阅读后续"最终决策"前，必须知道本报告的边界。

### 5.1 数据采集层面的局限

1. **6 个 7 日数据样本不足以验证"AIDSO 平均加权算法"**。需要至少 30 个样本验证规律稳定性。
2. **20 个领域的前 100 词未采集**。仅看前 20 词可能错过商机词。私人银行 67% 的商机词占比如果扩到 100 词可能下降。
3. **个人 IP 成交链路未深挖**。"月小妹儿"、"姚凯飞"等 IP 是否有私域成交链路（个人微信号、企微社群）未验证。
4. **同行买词占比未量化**。"同行买词"标签意味着竞争对手投流，但每个领域具体比例未统计。
5. **字数与商机词关系未验证**。直觉是字数长=商机词概率高，但未用 20 领域数据验证。
6. **概念股词追踪未做**。生命科学品类有"概念股"标签词，背后用户是投资人还是消费者未分。
7. **地域词分布未深挖**。三代试管/国际学校的地域词分布需要更多采集。

### 5.2 判断层面的局限

1. **5%/15%/30% 的商机词阈值是经验值**，需要至少 30 个真实成交案例验证。
2. **"私人银行是唯一高成交意图"是基于前 20 词判断**，可能漏掉其他品类。
3. **早癌筛查/高端体检的入口词修正（胃肠镜/全身体检）是基于本次新验证**，但可能还有其他口语化入口词未穷尽。
4. **海外辅助生殖的"真荒漠"判断未穷尽所有国家词**，俄罗斯/格鲁吉亚/乌克兰/哈萨克斯坦等具体国家词未测试。
5. **6.7-6.9 节"试管婴儿长大后问题 06-05 尖峰对应爆款视频"未反查**，是否真的对应一条爆款视频是猜测。

### 5.3 决策层面的局限

1. **3 个值得投入领域是基于现有数据的主观推荐**，没有历史案例验证 ROI。
2. **第六部分 ROI 计算全是未经验证的假设估算**——0.5% 抖音→线下转化率、0.1% 抖音→私域转化率都是从哪来的？没有数据支撑。**这是本报告最严重的局限性。**
3. **每个领域的"内容姿势"是经验判断**，没有真实跑过内容测试。

## 第七部分：最终结论

### 7.1 三个核心结论

**结论 1（✓ 数据支撑）**：20 个高客单价领域里，**私人银行是唯一有真高成交意图的品类**（商机词占比 67%）。其余都是种草型（试管婴儿/三代试管/ICL/国际学校/胃肠镜/全身体检）或事件泡沫型（劳力士/长江商学院/南极旅游等）。

**结论 2（✓ 数据支撑）**：AIDSO 月覆盖不能单独作为决策依据。174.61w 月覆盖的"全红婵戴劳力士"实际 7 日搜索只有 0-3 次。**必须用 7 日精确数据过滤"还活着的词"。**

**结论 3（✓ 部分数据支撑）**：原本归类为"荒漠"的早癌筛查、高端体检，换口语化入口词（胃肠镜、全身体检）后变成 33-60w 月覆盖的真大品类。**入口词选择决定能否发现抖音上的真实需求**。

### 7.2 三句话总结

抖音 20 个高客单价领域里只有私人银行有真高成交意图（67%），其余都是种草型或事件泡沫型。早癌筛查和高端体检的"荒漠"判断是错的，真入口词是胃肠镜和全身体检。**3 个值得现在投入的领域是试管婴儿、私人银行、三代试管+胃肠镜/全身体检（修正入口词后的真实品类），但 ROI 数据全是未经验证的假设估算**。

### 7.3 报告核心交付物

- 一套可复用的 7 日精确数据采集方法（hover + evaluate，已写入 010 第 24 章）
- 20 个领域的二维商业化成熟度评分矩阵（X=月覆盖档，Y=商机词占比档）

- 早癌筛查/高端体检的入口词修正（胃肠镜/全身体检）
