# 商业价值打分标准

> 这是关键词管理的**控制层规则**，不是搜索量、成交率或用户净值的事实数据。自动评分用于排序、刷新与归档，允许人工覆盖。

## 一、两条轴

| 轴 | 1分 | 3分 | 5分 |
|---|---|---|---|
| **距离成交环节** | 认知启蒙：靠谱吗、是什么、有什么区别 | 场景规划：养老、教育、传承、货币 | 成交动作：投保、优惠、门槛、提领、计划书、缴费 |
| **潜在资产/客单规模** | 客单不明确的泛保险词 | 储蓄险、分红险、年金等中产配置 | IUL、保费融资、杠杆寿、家族信托、大额保单 |

通常由两轴相加得到 **2～10分**；命中导航词、过期月份词或明显泛标签时直接降为 **1分**。以下示例均与当前评分器实际输出一致。

## 二、四象限

|  | **资产/客单较低** | **资产/客单较高** |
|---|---|---|
| **距离成交较远** | 长教育周期：`香港保险靠谱吗`、`香港保险有没有猫腻`、`内地保险和港险有什么区别` | 高净值教育：`IUL是什么`、`家族信托的作用` |
| **距离成交较近** | 大众成交：`香港保险怎么投保`、`香港重疾险优惠` | 高价值成交：`保费融资门槛`、`IUL计划书`、`杠杆寿保单贷款` |

## 三、10个层级

| 分数 | 特点 | 示例 |
|---:|---|---|
| **10** | 高净值且出现明确成交动作 | `保费融资投保门槛`、`IUL计划书`、`杠杆寿保单贷款` |
| **9** | 高净值方案已进入具体决策 | `香港家族信托哪家好`、`大额保单现金价值`、`多元货币保单提取` |
| **8** | 中高客单产品出现明确成交动作 | `友邦财富盈活门槛`、`香港保险金信托` |
| **7** | 一条轴很强，具备较明确商业机会 | `香港保险投保`、`香港保险优惠`、`香港保险提领`、`IUL` |
| **6** | 有明确配置场景或普通产品成交动作 | `香港保险养老金`、`港险提领密码` |
| **5** | 中段研究或中产产品泛兴趣 | `香港储蓄险`、`分红实现率怎么看` |
| **4** | 普通保障或低客单风险关注 | `香港重疾险`、`港险的汇率风险` |
| **3** | 长教育周期、转化慢，但仍与业务相关 | `什么人适合买港险`、`香港保险靠谱吗`、`香港保险有没有猫腻` |
| **2** | 只是在问具体产品“是什么”，资产和动作均不明确 | `AIA环宇盈活是什么产品` |
| **1** | 导航、过期、泛标签或明显低价值噪声 | `香港友邦保险公众号`、`香港保险6月优惠`、`港险干货` |

## 四、自动归档门槛

关键词必须**同时满足**以下条件才允许自动归档：

1. 已完成至少15天有效观察，从未跑过的词绝不归档；
2. 自动刷新频率已经降为15天；
3. 商业价值分 **≤3分**；
4. 近15天日均阅读增量中位数 **<10**；
5. 非人工锁定、非置顶，且没有刷新失败等待重试。

归档只改变状态，不删除历史快照、文章、关联词和评分证据。人工可以恢复，也可以锁定某个词永不自动归档。

## 五、评分边界

- “IUL=高净值”“保费融资=高客单”属于当前业务经验规则，不是系统观测事实；
- 分数只描述当前词面对应的商业优先级，不代表咨询一定成交；
- 真实热度仍由搜索结果、关联词、文章变化和阅读增量单独记录；
- 自动评分规则位于 `data/config/keyword_commercial_value_policy.json`，可随业务经验调整；
- 人工评分优先，自动重算不会覆盖人工分。
